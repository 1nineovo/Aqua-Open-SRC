Full ImGui Rendering

Experience a buttery smooth UI built entirely with Dear ImGui. Customizable, modern, and overlay-free rendering pipeline.

Extensive SDK

Comes with a comprehensive SDK wrapper for the Bedrock engine. Simplifies memory interaction and mod development significantly.

Clean Architecture

Modular and well-structured C++ codebase. Designed for readability, making it the perfect learning resource for internal cheat development.

Internal Hook

Direct memory hooking ensures minimal latency and maximum performance compared to external overlays. Optimized for 1.21.9x.

Build Environment

No pre-built binaries. Compile it yourself.
